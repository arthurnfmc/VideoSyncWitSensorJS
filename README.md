# Ferramenta de Sincronização Video/Sensor WIT

Essa ferramenta foi criada com o objetivo de sincronizar os dados obtidos em sensores WIT com vídeos. O seguinte documento prove uma explicação rápida sobre a instalação e utilização dessa ferramenta.

# Pré-requisitos

- [NodeJS e NPM](https://nodejs.org/pt) (v22.16.0 e 10.9.2)

# Alimentando os arquivos

- Dentro da pasta *public*, você encontrará as pastas *data* e *video*
- Na primeira, adicione os arquivos relativos ao sensor e, na outra, os arquivos dos vídeos

# Executando

- Estando na pasta principal do projeto, execute:

```
npm i
npm start
```

- e pronto! Um link para a aplicação será apresentado no terminal. Por padrão, este é http://localhost:5000
