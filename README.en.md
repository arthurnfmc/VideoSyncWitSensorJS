# WIT Video/Sensor Synchronization Tool

This tool was created to synchronize data captured from WIT sensors with videos. The following document provides a quick explanation on how to install and use this tool.

# Prerequisites

- [NodeJS and NPM](https://nodejs.org/) (v22.16.0 and 10.9.2)

# Adding Files

- Inside the public folder, you will find the data and video folders.
- In the first one, add the files related to the sensor; in the other, the video files.

# Running

- From the project's root folder, run:

```
npm i
npm start
```

- And that's it! A link to the application will be shown in the terminal. By default, it is http://localhost:5000
